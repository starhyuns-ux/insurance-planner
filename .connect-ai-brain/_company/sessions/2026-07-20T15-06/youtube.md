# 📺 레오 — YouTube 채널 분석(순위 1, 2, 3)

**YouTube 채널 분석(순위 1, 2, 3)**

📊 **Rank 1:** Recent Trend Analysis
---------------------------

* Goal: Analyze recent trends on YouTube and identify top-performing videos.
* Task: Run the `trend_sniper` tool to collect data on top-performing videos for the past 30 days.
* Output: A list of top-performing video titles, thumbnails, and descriptions.

📊 **Rank 2:** Channel Analytics
---------------------------

* Goal: Analyze YouTube channel performance metrics (e.g., views, engagement, etc.).
* Task: Run the `youtube_account` tool to collect channel analytics data.
* Output: A summary of channel performance metrics, including views, engagement, and more.

📊 **Rank 3:** Video Planning
---------------------------

* Goal: Plan new video content based on recent trends and channel analytics.
* Task: Use the `auto_planner` tool to generate a list of video ideas based on recent trends and channel analytics.
* Output: A list of video ideas, including titles, thumbnails, and descriptions.

**Assignments:**

* **레오 (Head of YouTube)**: Run the `trend_sniper` tool and collect data on top-performing videos for the past 30 days. Analyze the data and identify key trends.
* **Designer**: Create a list of video ideas based on recent trends and channel analytics using the `auto_planner` tool.
* **Writer**: Write captions and descriptions for the planned videos.

**Memo:**

* Recent trend analysis indicates that short-form videos with catchy thumbnails are performing well.
* Channel analytics show that engagement is highest during peak hours (e.g., 4-6 PM EST).
* Video planning should focus on creating engaging, bite-sized content with attention-grabbing titles and thumbnails.
